# AspectRatioEditor
This is an open-source Python script designed to handle and manage aspect ratios efficiently across different projects and applications. It was collaboratively developed by Glock30 and Lemmerly, providing a reliable and flexible solution for people who need precise control over screen and image dimensions.

# Other Things
This script is packed with helpful comments throughout to guide you, explain what each part does, and make it easier to understand and modify for your own projects.
